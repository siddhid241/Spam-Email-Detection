# Spam Email Detection using Machine Learning

## Description
This project classifies emails as spam or not spam using NLP techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Naive Bayes

## Dataset
SMS Spam Collection Dataset (Kaggle)

## How to Run
1. Open the notebook in Google Colab
2. Upload spam.csv
3. Run all cells
